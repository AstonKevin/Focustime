# 专注计时器

基于B站视频《为什么我能每天学习10小时》的核心理念开发的桌面应用，通过随机间隔的提示音帮助用户维持注意力，实现长时间高效学习。

## 功能特性

- **专注计时**: 设置专注时长，倒计时显示
- **随机提示音**: 设置最短-最长间隔，系统随机播放提示音
- **自定义提示音**: 支持选择本地音频文件（mp3、wav、ogg、m4a）
- **专注统计**: 记录每日专注时长和次数，最近7天图表展示

## 安装与运行

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

### 安装步骤

1. 克隆或下载项目
```bash
cd focus-timer
```

2. 安装依赖
```bash
npm install
```

3. 开发模式运行
```bash
npm run electron:dev
```

4. 打包构建
```bash
npm run electron:build
```

## 使用说明

### 基本使用

1. 启动应用后，默认显示专注计时页面
2. 点击「设置」可以调整专注时长和提示音区间
3. 点击「开始专注」启动计时器
4. 系统会在设定的随机间隔内播放提示音

### 设置说明

- **专注时长**: 设置每次专注的时间（分钟）
- **提示音区间**:
  - 间隔单位: 秒或分钟
  - 最短间隔: 提示音之间的最短时间
  - 最长间隔: 提示音之间的最长时间
- **提示音**: 选择自定义音频文件，或使用默认提示音

### 统计功能

- 自动记录每次专注完成的数据
- 显示今日专注总时长和次数
- 最近7天专注时长图表
- 最近10条专注记录

## 技术栈

- Electron: 桌面应用框架
- React: UI框架
- Vite: 构建工具
- electron-store: 本地数据存储

## 项目结构

```
focus-timer/
├── electron/           # Electron主进程
│   ├── main.js        # 主进程入口
│   └── preload.js     # 预加载脚本
├── src/               # React源码
│   ├── components/    # React组件
│   ├── hooks/         # 自定义Hooks
│   ├── store/         # 数据存储
│   └── styles/        # 样式文件
├── public/            # 静态资源
└── package.json       # 项目配置
```

## 注意事项

- 提示音区间设置需满足：最短间隔 < 最长间隔
- 自定义提示音支持格式：mp3、wav、ogg、m4a
- 统计数据本地存储，不会丢失
